Give us a Twitter Token

TraceMap is an open source web tool that allows you to visualize the spread of a tweet, analyze its trustworthiness and to engage with others in fighting misinformation. To achieve this, we need to gather plenty of data and make a lot of twitter API calls. Unfortunately, the number of twitter API calls is limited, so our data gathering depends on accumulating Twitter tokens. That’s how you can help us: by logging in once, you increase our data-transfer rate. We are only interested in public Twitter data, you are granting our app the most permissive read-only access.

Our app cannot access any private information and cannot send tweets.

What are Twitter Tokens?

Tokens are unique keys that allow external applications to request and get a certain amount of data from Twitter. Once a token has been used, it becomes inactive for a few minutes, limiting the amount of data one can actually access. For us to access the volume of data we need, we must have a large number of tokens.

Who uses tokens?

Apps that have a high traffic of people get more tokens and, therefore, access to more data. We need approximately 600 of them to maintain the public German data up-to-date and multiple thousands to scale up to Twitter's entire network. That's why your help is decisive!

How does TraceMap get Twitter tokens?

A token is generated after you click on the button below. If you are not yet logged in to Twitter, you will be asked to confirm your identity by typing in your Twitter login credentials. After confirming that you allow TraceMap to get access to Twitter data, this request will reach Twitter and the token will be given to us. Each Twitter user can donate only one token to TraceMap. You can nevertheless donate tokens to other apps that you support with one account. Also, you can always revoke access to a certain token going from the "Settings and Privacy” to the “Apps" menu inside Twitter and clicking on the "Revoke access" button corresponding to each app.
